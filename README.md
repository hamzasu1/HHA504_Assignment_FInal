# HHA504_Assignment_Final

Release of information (ROI) is a critical workflow in every hospital organization. This multi-service integration introduces storage compute, database, and serverless components. The security design includes Role based access control(RBAC) and PHI protection in accordance with HIPAA. The ROI tracking system demonstrates how cloud technologies can modernize manual processes. The cloud platform utilized in this project was Azure.
